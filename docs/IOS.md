# iOS 版本

> iOS 版本代码和完整文档在 [ios 分支](https://github.com/MinJung-Go/CompositionHelper/tree/ios)

## 快速开始

```bash
git clone -b ios https://github.com/MinJung-Go/CompositionHelper.git
cd CompositionHelper
open CompositionHelper.xcodeproj
```

## 前置要求

| 工具 | 最低版本 |
|------|---------|
| macOS | 12.0 (Monterey) |
| Xcode | 14.0 |
| iOS | 15.0 |
| Swift | 5.0 |

## 技术栈

- SwiftUI + Vision Framework + AVFoundation
- 无外部依赖
