# 功能详解

> CompositionHelper 支持的 19 种构图类型详细说明

## 经典构图（7种）

### 三分法 (Rule of Thirds)

**适用场景**: 风景、人像、街拍

将重要元素放在九宫格的四个交点上，适用于大多数摄影场景。
```
┌─────────┬─────────┬─────────┐
│         │         │         │
├─────────┼─────────┼─────────┤
│         │    *    │         │
├─────────┼─────────┼─────────┤
│         │         │         │
└─────────┴─────────┴─────────┘
```

### 中心构图 (Center Composition)

**适用场景**: 对称场景、特写、肖像

主体完全居中，适合对称场景或需要强调主体时。

### 对角线构图 (Diagonal)

**适用场景**: 运动场景、建筑、道路

主体沿对角线排列，创造动态感和深度感。

### 框架构图 (Frame within a Frame)

**适用场景**: 门框、窗户、拱门

利用前景元素形成框架，创造深度和层次感。

### 引导线构图 (Leading Lines)

**适用场景**: 道路、河流、桥梁

利用线条引导观众视线，创造三维空间感。

### S形曲线构图 (S-Curve)

**适用场景**: 河流、道路、海岸线

寻找自然的 S 形曲线，创造流动感和柔和感。

### 黄金螺旋 (Golden Spiral)

**适用场景**: 艺术创作、人像、自然

主体位于螺旋终点，辅助元素沿螺旋排列，基于黄金比例（1:1.618）。

## 现代构图（7种）

### 黄金三角 (Golden Triangle)

**适用场景**: 建筑、产品、人像

主体位于三角形顶点，利用对角线创造三角形。

### 对称构图 (Symmetry)

**适用场景**: 对称建筑、倒影、人脸

完全左右或上下对称，创造平衡和稳定感。

### 负空间 (Negative Space)

**适用场景**: 极简摄影、艺术创作

大面积留白（70%以上），主体占据小部分，创造简洁和焦点。

### 模式重复 (Pattern Repetition)

**适用场景**: 建筑、纹理、自然

寻找重复的元素，可以打破模式增加趣味。

### 隧道式 (Tunnel)

**适用场景**: 桥洞、走廊、门窗

创造向深处延伸的效果，主体在隧道尽头。

### 分割构图 (Split)

**适用场景**: 多主题、对比、双重曝光

将画面分为两个或多个部分，创造对比或互补效果。

### 透视焦点 (Vanishing Point)

**适用场景**: 道路、建筑、城市

利用透视线汇聚于一点，创造强烈的空间感。

---

## 视角构图（5种）

### 隐形线 (Implied Line)

**适用场景**: 人物视线、手势、方向

利用视线或手势创造隐形线条，引导观众看向主体。

### 充满画面 (Fill the Frame)

**适用场景**: 微距、纹理、特写

主体填充画面 85% 以上，消除背景干扰，强调细节和质感。

### 低角度 (Low Angle)

**适用场景**: 建筑仰视、运动、人像

相机低于主体向上拍摄，创造夸张的透视效果。

### 高角度 (High Angle)

**适用场景**: 风景俯视、美食、布局

相机高于主体向下拍摄，展示全局和布局。

### 深度层次 (Depth Layer)

**适用场景**: 风景、街景、建筑

将画面分为前景、中景、背景三层，创造丰富的空间层次感。

## 构图选择建议

### 根据场景选择

| 场景 | 推荐构图 |
|------|----------|
| 风景 | 三分法、引导线、S形曲线、深度层次 |
| 人像 | 三分法、中心构图、负空间 |
| 建筑 | 对称构图、透视焦点、黄金三角、低角度 |
| 街拍 | 对角线、框架构图、充满画面 |
| 微距 | 充满画面、中心构图 |
| 美食 | 高角度、低角度、充满画面 |

### 根据情感选择

| 情感 | 推荐构图 |
|------|----------|
| 平静 | 三分法、对称构图 |
| 动感 | 对角线、引导线、S形曲线 |
| 力量 | 低角度、充满画面 |
| 简洁 | 负空间、中心构图 |
| 神秘 | 框架构图、隧道式 |
| 深度 | 深度层次、透视焦点 |
