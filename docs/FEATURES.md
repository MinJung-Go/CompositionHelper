# 构图类型详解

> CompositionHelper 支持的 19 种构图类型，分为经典 / 现代 / 视角三大分类

---

## 经典构图（7 种）

### 三分法 (Rule of Thirds)

**适用场景**: 风景、人像、街拍

将画面等分为九宫格，主体置于四个交叉点上，是最通用的构图法则。

```
┌─────────┬─────────┬─────────┐
│         │         │         │
├─────────┼─────────┼─────────┤
│         │    *    │         │
├─────────┼─────────┼─────────┤
│         │         │         │
└─────────┴─────────┴─────────┘
```

**技巧**: 地平线放在上/下三分之一处，主体放在交叉点上。

---

### 中心构图 (Center)

**适用场景**: 对称场景、特写、肖像

主体完全居中，强调主体本身。适合背景简洁或需要突出单一主体的场景。

```
┌─────────────────────────────┐
│                             │
│            ┌─────┐          │
│            │  *  │          │
│            └─────┘          │
│                             │
└─────────────────────────────┘
```

**技巧**: 背景越简洁效果越好，避免干扰元素。

---

### 对角线 (Diagonal)

**适用场景**: 运动场景、建筑、道路

主体沿对角线排列，增强动感和深度。

```
┌─────────────────────────────┐
│ *                           │
│   \                         │
│     \                       │
│       \                     │
│         *                   │
└─────────────────────────────┘
```

**技巧**: 适合表现运动方向或空间延伸感。

---

### 框架构图 (Frame within a Frame)

**适用场景**: 门框、窗户、拱门

利用前景元素形成天然框架，引导视线聚焦到主体上。

```
┌─────────────────────────────┐
│ ╔═════════════════════════╗ │
│ ║                         ║ │
│ ║           *             ║ │
│ ║                         ║ │
│ ╚═════════════════════════╝ │
└─────────────────────────────┘
```

**技巧**: 门窗、树枝、拱形建筑都是天然框架。

---

### 引导线 (Leading Lines)

**适用场景**: 道路、河流、桥梁

利用场景中的线条引导观众视线移向主体。

```
┌─────────────────────────────┐
│                     /       │
│                   /         │
│     *           /           │
│               /             │
└─────────────────────────────┘
```

**技巧**: 线条应指向主体或消失点，创造三维空间感。

---

### S 形曲线 (S-Curve)

**适用场景**: 河流、道路、海岸线

画面中包含 S 形曲线，创造流动感和柔美自然的视觉效果。

```
┌─────────────────────────────┐
│       ~~~              *    │
│    ~        ~               │
│  ~              ~           │
│                    ~        │
└─────────────────────────────┘
```

**技巧**: 寻找自然的 S 形元素，适合风景摄影。

---

### 黄金螺旋 (Golden Spiral)

**适用场景**: 艺术创作、人像、自然

基于黄金比例 (1:1.618) 的螺旋线，主体位于螺旋焦点处。

```
┌─────────────────────────────┐
│                    o        │
│               o             │
│            *                │
│        o                    │
│    o                        │
└─────────────────────────────┘
```

**技巧**: 辅助元素沿螺旋线排列，主体在螺旋终点。

---

## 现代构图（7 种）

### 黄金三角 (Golden Triangle)

**适用场景**: 建筑、产品、人像

画面被对角线和垂线分割为黄金比例的三角形区域。

```
┌─────────────────────────────┐
│ *                         / │
│   \                     /   │
│     \                 /     │
│       \             /       │
└─────────────────────────────┘
```

**技巧**: 主体放在三角形顶点或对角线交叉处。

---

### 对称构图 (Symmetry)

**适用场景**: 对称建筑、倒影、人脸

完全左右或上下对称，创造平衡稳定的视觉效果。

```
┌─────────────────────────────┐
│         |       |           │
│         |   *   |           │
│         |       |           │
└─────────────────────────────┘
```

**技巧**: 水面倒影、对称建筑是最佳应用场景。

---

### 负空间 (Negative Space)

**适用场景**: 极简摄影、艺术创作

大面积留白 (70%+)，主体占据小部分画面，突出主体的同时营造简洁氛围。

```
┌─────────────────────────────┐
│                             │
│                      *      │
│                             │
│                             │
└─────────────────────────────┘
```

**技巧**: 留白越多，主体越突出。

---

### 模式重复 (Pattern Repetition)

**适用场景**: 建筑、纹理、自然

利用重复出现的元素创造节奏和视觉韵律，可以通过打破模式增加趣味。

```
┌─────────────────────────────┐
│  o  o  o  *  o  o  o       │
│  o  o  o  o  o  o  o       │
│  o  o  o  o  o  o  o       │
└─────────────────────────────┘
```

**技巧**: 寻找重复元素，在其中放入一个"例外"制造焦点。

---

### 隧道式 (Tunnel)

**适用场景**: 桥洞、走廊、门窗

创造向深处延伸的效果，主体位于隧道尽头。

```
┌─────────────────────────────┐
│    ╔═════════════════╗      │
│    ║                 ║      │
│    ║       *         ║      │
│    ╚═════════════════╝      │
└─────────────────────────────┘
```

**技巧**: 利用透视缩小营造空间纵深感。

---

### 分割构图 (Split)

**适用场景**: 多主题、对比、双重曝光

将画面分为两个或多个部分，利用颜色、明暗或内容分割画面创造对比效果。

```
┌─────────────────────────────┐
│ ################            │
│ ################            │
│ ################            │
└─────────────────────────────┘
```

**技巧**: 可用颜色、明暗、水面等自然元素来分割。

---

### 透视焦点 (Vanishing Point)

**适用场景**: 道路、建筑、城市

利用透视线汇聚于消失点，创造强烈的空间纵深感。

```
┌─────────────────────────────┐
│  /    /    /    /    /      │
│/    /    /    *    /        │
│   /    /    /    /          │
└─────────────────────────────┘
```

**技巧**: 消失点可以居中也可以偏移，指向主体最佳。

---

## 视角构图（5 种）

### 隐形线 (Implied Line)

**适用场景**: 人物视线、手势、方向

利用视线方向或手势等创造隐形线条，引导观众视线。

```
┌─────────────────────────────┐
│  A  ──────────────>  *      │
│                             │
│                             │
└─────────────────────────────┘
```

**技巧**: 人物视线方向、手指指向都是天然的隐形引导线。

---

### 充满画面 (Fill the Frame)

**适用场景**: 微距、纹理、特写

主体填充画面 85% 以上，消除背景干扰，强调细节和质感。

```
┌─────────────────────────────┐
│#############################│
│#############################│
│############*################│
│#############################│
└─────────────────────────────┘
```

**技巧**: 靠近主体拍摄，或裁切画面。

---

### 低角度 (Low Angle)

**适用场景**: 建筑仰视、运动、人像

相机低于主体向上拍摄，创造夸张透视，主体显得更高大有力。

```
┌─────────────────────────────┐
│              |              │
│              |              │
│              *              │
│             / \             │
└─────────────────────────────┘
```

**技巧**: 配合广角镜头效果更明显。

---

### 高角度 (High Angle)

**适用场景**: 风景俯视、美食、布局

相机高于主体向下拍摄，展示全局和布局。

```
┌─────────────────────────────┐
│            / \              │
│           *   *             │
│          *     *            │
└─────────────────────────────┘
```

**技巧**: 适合展示食物摆盘、建筑全貌、群体场景。

---

### 深度层次 (Depth Layer)

**适用场景**: 风景、街拍、人像

通过前景、中景、背景三个层次创造画面纵深感，增强立体效果。

```
┌─────────────────────────────┐
│  (远景)                     │
│         [中景 *]            │
│  <<<前景>>>                 │
└─────────────────────────────┘
```

**技巧**: 前景提供框架或引导，中景放主体，背景提供环境信息。

## 构图选择参考

### 按拍摄场景

| 场景 | 推荐构图 |
|------|---------|
| 风景 | 三分法、引导线、S 形曲线、深度层次 |
| 人像 | 三分法、中心构图、负空间、框架构图 |
| 建筑 | 对称构图、透视焦点、黄金三角、低角度 |
| 街拍 | 对角线、框架构图、充满画面、深度层次 |
| 微距 | 充满画面、中心构图 |
| 美食 | 高角度、充满画面、对称构图 |

### 按表达情感

| 情感 | 推荐构图 |
|------|---------|
| 平静稳定 | 三分法、对称构图、中心构图 |
| 动感活力 | 对角线、引导线、S 形曲线 |
| 力量震撼 | 低角度、充满画面、隧道式 |
| 简洁优雅 | 负空间、中心构图 |
| 空间纵深 | 透视焦点、深度层次、隧道式 |
