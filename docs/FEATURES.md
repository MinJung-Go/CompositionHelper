# 功能详解

> CompositionHelper 支持的 18 种构图类型详细说明

## 经典构图（7种）

### 📐 三分法 (Rule of Thirds)

**适用场景**: 风景、人像、街拍

**效果描述**: 经典平衡，主体位于九宫格交点

**使用技巧**:
- 将重要元素放在九宫格的四个交点上
- 适用于大多数摄影场景
- 地平线可以放在上或下三分之一处

**示例**:
```
┌─────────┬─────────┬─────────┐
│         │         │         │
├─────────┼─────────┼─────────┤
│         │    ●    │         │
├─────────┼─────────┼─────────┤
│         │         │         │
└─────────┴─────────┴─────────┘
```

---

### 🎯 中心构图 (Center Composition)

**适用场景**: 对称场景、特写、肖像

**效果描述**: 强调主体，居中对称

**使用技巧**:
- 主体完全居中
- 适合对称场景或需要强调主体时
- 背景应简洁，避免干扰

**示例**:
```
┌─────────────────────────────┐
│                             │
│            ┌─────┐          │
│            │  ●  │          │
│            └─────┘          │
│                             │
└─────────────────────────────┘
```

---

### ➡️ 对角线构图 (Diagonal)

**适用场景**: 运动场景、建筑、道路

**效果描述**: 增强动感，引导视线流动

**使用技巧**:
- 主体沿对角线排列
- 创造动态感和深度感
- 适合表现运动或延伸感

**示例**:
```
┌─────────────────────────────┐
│                         ●   │
│                     ●       │
│                 ●           │
│             ●               │
└─────────────────────────────┘
```

---

### 🖼️ 框架构图 (Frame within a Frame)

**适用场景**: 门框、窗户、拱门

**效果描述**: 突出主体，创造层次感

**使用技巧**:
- 利用前景元素形成框架
- 主体在框架内或框架边缘
- 创造深度和层次感

**示例**:
```
┌─────────────────────────────┐
│ ╔═════════════════════════╗ │
│ ║                         ║ │
│ ║           ●             ║ │
│ ║                         ║ │
│ ╚═════════════════════════╝ │
└─────────────────────────────┘
```

---

### 🛤️ 引导线构图 (Leading Lines)

**适用场景**: 道路、河流、桥梁

**效果描述**: 引导视线，增强深度

**使用技巧**:
- 利用线条引导观众视线
- 线条应指向主体
- 创造三维空间感

**示例**:
```
┌─────────────────────────────┐
│                     /       │
│                   /         │
│     ●           /           │
│               /             │
└─────────────────────────────┘
```

---

### 〰️ S形曲线构图 (S-Curve)

**适用场景**: 河流、道路、海岸线

**效果描述**: 柔美流畅，自然美感

**使用技巧**:
- 寻找自然的 S 形曲线
- 创造流动感和柔和感
- 适合风景摄影

**示例**:
```
┌─────────────────────────────┐
│                             │
│       ~~~              ●    │
│    ~        ~               │
│  ~              ~           │
└─────────────────────────────┘
```

---

### 🌀 黄金螺旋 (Golden Spiral)

**适用场景**: 艺术创作、人像、自然

**效果描述**: 自然比例，黄金分割

**使用技巧**:
- 主体位于螺旋终点
- 辅助元素沿螺旋排列
- 基于黄金比例（1:1.618）

**示例**:
```
┌─────────────────────────────┐
│                    ○        │
│               ○             │
│            ●                │
│        ○                    │
│    ○                        │
└─────────────────────────────┘
```

## 现代热门构图（11种）

### 🔺 黄金三角 (Golden Triangle)

**适用场景**: 建筑、产品、人像

**效果描述**: 黄金比例三角形

**使用技巧**:
- 主体位于三角形顶点
- 利用对角线创造三角形
- 适合建筑和产品摄影

**示例**:
```
┌─────────────────────────────┐
│ ●                           │
│   ╱                         │
│  ╱                          │
│ ╱                           │
└─────────────────────────────┘
```

---

### 🔄 对称构图 (Symmetry)

**适用场景**: 对称建筑、倒影、人脸

**效果描述**: 平衡有序，稳定感

**使用技巧**:
- 完全左右或上下对称
- 主体居中
- 创造平衡和稳定感

**示例**:
```
┌─────────────────────────────┐
│         │       │         │
│         │   ●   │         │
│         │       │         │
└─────────────────────────────┘
```

---

### ⬜ 负空间 (Negative Space)

**适用场景**: 极简摄影、艺术创作

**效果描述**: 突出主体，留白艺术

**使用技巧**:
- 大面积留白（70%以上）
- 主体占据小部分
- 创造简洁和焦点

**示例**:
```
┌─────────────────────────────┐
│                             │
│                      ●      │
│                             │
│                             │
└─────────────────────────────┘
```

---

### 🔁 模式重复 (Pattern Repetition)

**适用场景**: 建筑、纹理、自然

**效果描述**: 节奏感，视觉韵律

**使用技巧**:
- 寻找重复的元素
- 可以打破模式增加趣味
- 创造节奏和视觉韵律

**示例**:
```
┌─────────────────────────────┐
│  □  □  □  ●  □  □  □       │
│  □  □  □  □  □  □  □       │
│  □  □  □  □  □  □  □       │
└─────────────────────────────┘
```

---

### 🚇 隧道式 (Tunnel)

**适用场景**: 桥洞、走廊、门窗

**效果描述**: 深度感，透视效果

**使用技巧**:
- 创造向深处延伸的效果
- 主体在隧道尽头
- 强烈的空间感

**示例**:
```
┌─────────────────────────────┐
│    ╔═════════════════╗      │
│    ║                 ║      │
│    ║       ●         ║      │
│    ║                 ║      │
│    ╚═════════════════╝      │
└─────────────────────────────┘
```

### 📊 分割构图 (Split)

**适用场景**: 多主题、对比、双重曝光

**效果描述**: 对比效果，分割画面

**使用技巧**:
- 将画面分为两个或多个部分
- 创造对比或互补效果
- 可以使用颜色、明暗分割

```
│████████████████░░░░░░░░░░░░░│
│████████████████░░░░░░░░░░░░░│
└─────────────────────────────┘
```

### 🔍 透视焦点 (Vanishing Point)

**适用场景**: 道路、建筑、城市

**效果描述**: 空间感，消失点

**使用技巧**:
- 利用透视线汇聚于一点
- 创造强烈的空间感
- 消失点可以居中或偏移

**示例**:
```
┌─────────────────────────────┐
│  ╱    ╱    ╱    ╱    ╱      │
│╱    ╱    ╱    ●    ╱        │
│   ╱    ╱    ╱    ╱          │
└─────────────────────────────┘
```

## 构图选择建议

### 根据场景选择

| 场景 | 推荐构图 |
|------|----------|
| 风景 | 三分法、引导线、S形曲线 |
| 人像 | 三分法、中心构图、负空间 |
| 建筑 | 对称构图、透视焦点、黄金三角 |
| 街拍 | 对角线、框架构图、充满画面 |
| 微距 | 充满画面、中心构图 |
| 美食 | 高角度、低角度、充满画面 |

### 根据情感选择

| 情感 | 推荐构图 |
|------|----------|
| 平静 | 三分法、对称构图 |
| 动感 | 对角线、引导线、S形曲线 |
| 力量 | 低角度、充满画面 |
| 简洁 | 负空间、中心构图 |
| 神秘 | 框架构图、隧道式 |
